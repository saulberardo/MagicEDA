********************
Install Instructions
********************

.. installing_with_pip:

Installing with PIP
===================

Just type this::

  pip install magikeda


.. installing_from_source:

Installing from Source
=========================



Just type this::

  git clone https://github.com/saulberardo/MagikEDA.git magikeda
  cd magikeda
  python setup.py install
